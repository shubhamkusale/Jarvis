# 🤖 Jarvis — AI Assistant Development Journey

## Overview

Jarvis is a personal AI assistant project built step-by-step over 32 weeks as part of my AI engineering learning journey.

The goal is to progressively evolve Jarvis from a simple rule-based chatbot into a full intelligent assistant with voice interaction, memory, and AI decision-making capabilities.

---

## 🚀 Current Version

**Week 1 — Jarvis V0.1**

Features:

* Text-based conversation
* Rule-based responses
* Time query support
* Exit commands
* Continuous conversation loop

This version focuses on learning core programming concepts and assistant architecture.

---

## 🧠 Architecture Concept

Jarvis follows a simple pipeline:

User Input → Processing → Response → Loop

This architecture will expand in future weeks to include:

* Voice input/output
* Memory module
* NLP models
* AI agents
* Automation tools
